## lab1-1_calculator
깃 허브 연습

```python
def add(x,y):
  return x + y
def subtract:
  return x - y
  
 # ******몸체*************
 
 print("Select operation")
 print("1.Add")
 print("2.Subtract")
 
 # Take input from the user
 
 choice = input("Enter choice(1/2)")
 
 num1 = float(input("첫번째 숫자를 입력하시오: "))
 num2 = float(input("두번째 숫자를 입력하시오: "))
 
 if choice == "1":
   print(num1,"+",num2,"=",add(num1,num2))
 elif choice == "2":
   print(num1,"-",num2,"=",subtract(num1,num2))
 else:
  print("틀린 입력을 하였습니다.")
  
```
